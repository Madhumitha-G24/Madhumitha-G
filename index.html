Projects

Madee Café

A full-stack web application built using Django and MySQL, designed for a smooth online food ordering experience. The platform includes features like dynamic menu rendering, a shopping cart system, and Google Pay / Cash on Delivery (GPay/COD) integration for payments. It also features an admin dashboard to manage orders and customer data. The focus was on creating a responsive UI, efficient data storage, and seamless backend integration.

Technologies: Django, Python, HTML5, CSS3, JavaScript, MySQL, Bootstrap

Automated Bus Scheduler

A smart scheduling system created for the Delhi Transport Corporation (DTC). It automates bus route and time allocation by analyzing real-time traffic, passenger load, and route history. The solution included a dynamic algorithm to adjust schedules and dispatch buses accordingly. It focused on minimizing wait time and optimizing route efficiency.

Technologies: Python, Django, Google Maps API, HTML/CSS, JavaScript, MySQL

Smart Parking Solution

Developed during the TANCAM Hackathon, this IoT-enabled smart parking system allows users to view and book parking slots in real-time. The system includes a web dashboard, sensor integration for slot availability detection, and navigation guidance to the nearest available space. The backend was designed to handle concurrency and real-time updates efficiently.

Technologies: HTML, CSS, JavaScript, Python, Django, IoT Sensors, Bootstrap
